WORKPIECE HEIGHT UTILITY

This program probes 2 user specified locations in the Z axis and calculates the difference in heights.

  • when checked, the SET buttons are enabled
  • this allows the user to automatically fill in the X, Y and Z parameters with the current position as displayed on the DROs.
  • when checked, the calculated height is automatically transferred to the Workpiece Height field in the main screen.
  • the X, Y and Z coordinates specify where the first probing routine should start, in current WCS
  • the X, Y and Z coordinates specify where the second probing routine should start, in current WCS

Parameter Validation

Before running the probe routines, probe parameters are checked to make sure no problems will occur. Any 2 points within the machine operating volume can be specified.

The units are set to whatever the machine units are defined as in the INI file.

  • search velocity must be greater than 0.0
  • probe velocity must be greater than 0.0
  • maximum probe distance must be greater than 0.0
  • retract distance must be greater than 0.0 and less than maximum probe distance
  • Z safe height must be greater than P1 Z height
  • P1 Z height must be greater than P2 Z height
  • probe parameters are read from SETTINGS -> PROBE and checked for validity
  • if any errors are detected while checking validity, the probe routine will not run
  • any errors detected will be highlighted with red text in SETTINGS -> PROBE
  • the machine will jog to the first location and then probe down and record the Z1 position
  • the machine then jogs to the second location and probes down and records the Z2 position
  • the difference in probed values is reported as Calculated Workpiece Height
Warning
 Setting incorrect values can lead to crashes into fixtures on the machine work surface. Initial testing with no tool and safe heights is recommended.

  • causes all jog and probe routines currently executing to stop
