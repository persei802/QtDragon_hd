BASIC PROBE EXAMPLE

Demo of inside corner probing using the top right button in Basic Probe Inside Measurements tab.

While all probe entries must be correct, the most important settings to change for each probe sequence are:

  • STEP OFF - distance away from edge before rough probing
  • Z CLEARANCE - distance from probe to top of material
  • EXTRA DEPTH - distance to lower probe from top of material
  • EDGE WIDTH - distance along edge wall (away from corner) to start probing

NOTE : These distances are always to be set in 'machine units' (mm for metric machine, inch for imperial machine)

Manually set the probe at the intersection of the edges (ie corner) of material as described by the purple dot on the button. Set it to Z CLEARANCE above the top of material. These can be done by eye.

Set EXTRA DEPTH to a value that you want the probe to go below the toptop of material. (So the probe will move from it's start position down Z Clearance + Extra Depth distance)

Set STEP OFF to a value that definitely gives clearance from the wall so that when the probe goes down, it doesn't hit anything.

Set EDGE WIDTH to a value that describes the distance measured from the corner, along the wall to where you wish to probe. This edge distance will be used along the X wall and then the Y wall.

Sequence after pressing the probe button:

  • rapid to EDGE WIDTH distance away from corner at the same time moving STEP OFF away from edge. So this is a slightly diagonal move.
  • move probe down by Z CLEARANCE + EXTRA DEPTH + PROBE DIA (probe tip diameter)
  • probe wall twice (rough and fine)
  • move diagonally to the other wall as set by EDGE WIDTH and STEP OFF
  • probe wall twice (rough and fine)
  • raise probe up by Z CLEARANCE + EXTRA DEPTH + PROBE DIA (returns to starting height)
  • rapid back to calculated corner
  • if auto zero button is enabled, set X and Y of the current user system to zero
